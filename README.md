# foruns-de-votacao
**Configurações**
>Instale o pipenv se vc não tiver instalado:
>
>pip install pipenv

Configurações:

  ```
  pipenv install
  export FLASK_APP=app.py
  ```
  >se quiser o modo debug:
  >
  >export FLASK_DEBUG=1


  **Exceutar**

  ```
  flask run
  ```

  >no navegador coloque a url:
  >
  >127.0.0.1:5000/
